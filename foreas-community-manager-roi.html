Le Cerveau d'Optimisation Continue

Boucle de rétroaction autonome — chaque publication rend la suivante meilleure

COLLECTE
Aspiration des métriques par réseau
n8n récupère automatiquement via les APIs de chaque plateforme : vues, likes, partages, commentaires, saves, watch time, taux de complétion vidéo, clics lien bio, nouveaux followers, heure de pic d'engagement. Stocké dans Supabase (table analytics_social).
ANALYSE
Pattern detection par Claude API
Chaque semaine, Claude analyse l'ensemble des données collectées et identifie : quel type de contenu performe le mieux (vidéo vs image, question vs statement, stat vs story), quels horaires maximisent l'engagement par plateforme, quels mots-clés et accroches déclenchent le plus d'interactions, quel ton fonctionne (inspirant vs informatif vs provoc).
SCORING
Score de performance par catégorie de contenu
Chaque publication reçoit un score composite (engagement_rate × reach × conversion_bio_click). Les contenus sont classés par archétype : "stat chauffeur", "actu VTC", "behind the scenes", "feature Ajnaya", "témoignage", "provoc marché". Le cerveau identifie les archétypes gagnants et les perdants.
ADAPTATION
Mise à jour automatique du prompt system
Le rapport d'analyse met à jour les instructions du community manager IA : augmenter la fréquence des archétypes gagnants, réduire ou éliminer les perdants, ajuster les horaires de publication, affiner le ton et les accroches. Le prompt system évolue chaque semaine sans intervention humaine.
GÉNÉRATION
Contenu optimisé publié automatiquement
La semaine suivante, le contenu généré intègre tous les apprentissages. Le cycle recommence. Chaque itération affine la compréhension de l'audience FOREAS et maximise la portée organique.
↻ Boucle infinie d'amélioration
Le système ne stagne jamais. Plus il publie, plus il apprend. Plus il apprend, mieux il performe. Effet composé sur la portée organique.

Signaux captés par le cerveau

Chaque métrique oriente la prochaine décision éditoriale

Watch time moyen
Durée moyenne de visionnage par vidéo. Détecte si l'accroche retient ou si le contenu perd l'audience.
→ Ajuste la durée et le hook des 3 premières secondes
Taux de save/bookmark
Le signal le plus fort sur Instagram et TikTok. Indique un contenu à haute valeur perçue.
→ Amplifie les formats "valeur actionnable"
Ratio partage/vue
Mesure la viralité intrinsèque. Un partage vaut 10x un like en portée organique.
→ Priorise les contenus "shareables" (stats choc, comparaisons)
Heure de pic
Identifie les fenêtres d'engagement max par plateforme, par jour de semaine.
→ Ajuste le scheduling automatiquement
Sentiment commentaires
Analyse NLP des commentaires pour détecter enthousiasme, scepticisme, questions récurrentes.
→ Génère du contenu qui répond aux objections détectées
Taux de clic bio
Conversion sociale → site/app. Le KPI business ultime du community management.
→ Optimise les CTA et le placement du lien
Croissance followers
Vélocité de gain par réseau. Corrélée aux types de contenu publiés la semaine précédente.
→ Identifie les "growth triggers" par plateforme
Cross-plateforme
Même contenu adapté : compare les performances entre réseaux pour identifier où chaque format performe le mieux.
→ Route chaque format vers son réseau optimal
